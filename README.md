# NorthStar Databases and Analytics Assignment

This repository contains the Google Colab notebooks for the NorthStar Urban Mobility and Logistics case study.

## Tools Used

- R: DBI, RSQLite, dplyr, ggplot2
- Python: Pandas, NumPy, Matplotlib, PyMongo
- MongoDB Atlas
- Google Colab

## Notebooks

1. SQL in R analysis
2. R analytics and statistical tests
3. Python data processing and visualisation
4. MongoDB Atlas design, CRUD, aggregation and indexing

## Reproducibility

Run the notebooks in order.

MongoDB Atlas credentials are not included for security reasons.
